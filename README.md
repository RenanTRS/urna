## Getting Started
### Installing
**Cloning the repository**
```
$git clone https://github.com/RenanTRS/urna.git
$cd urna
```

**Installing dependencies**
```
$yarn
```
or 
```
$npm install
```

## Running
With all dependencies installed and the environment properly configured, you cannow run the project:
```
$yarn start
```
or
```
$npm start
```
